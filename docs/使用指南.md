## 二、轴强度计算（ShaftCalculator）

`ShaftCalculator` 提供轴系强度计算功能，支持多种材料和计算场景。

### 2.1 初始化计算器

```python
from calculations.shaft import ShaftCalculator

# 默认材料 45钢
calc = ShaftCalculator()

# 指定其他材料
calc_40cr = ShaftCalculator(material="40Cr")
calc_q235 = ShaftCalculator(material="Q235")
```

**支持的材料**：45、40Cr、Q235、20Cr

### 2.2 截面几何计算

```python
diameter = 30.0  # mm

# 极惯性矩 J_p = π·d⁴/32
J_p = calc.polar_moment_of_inertia(diameter)

# 轴向惯性矩 I_y = π·d⁴/64
I_y = calc.axial_moment_of_inertia(diameter)

# 抗弯截面系数 W = π·d³/32
W = calc.section_modulus(diameter)

# 抗扭截面系数 W_p = π·d³/16
W_p = calc.torsion_modulus(diameter)
```

### 2.3 轴径估算

#### 按扭转强度估算

适用于传动轴（主要承受扭矩）：

```python
# 已知扭矩 T = 100000 N·mm
torque = 100000  # N·mm
allowable_shear = 40  # MPa，许用剪切应力

diameter = calc.estimate_diameter_by_torsion(
    torque=torque,
    allowable_shear_stress=allowable_shear
)
# 自动圆整到 GB/T 2822 标准直径
```

#### 按弯扭组合强度估算

适用于转轴（同时承受弯矩和扭矩）：

```python
torque = 100000       # N·mm
bending_moment = 50000  # N·mm
allowable_stress = 60  # MPa

diameter = calc.estimate_diameter_by_bending_torsion(
    torque=torque,
    bending_moment=bending_moment,
    allowable_stress=allowable_stress
)
```

### 2.4 承载能力校核

#### 许用扭矩计算

```python
diameter = 30.0  # mm

# 使用材料默认许用应力（剪切强度的1/3）
allowable_torque = calc.calculate_allowable_torque(diameter)

# 指定许用剪切应力
allowable_torque = calc.calculate_allowable_torque(
    diameter=diameter,
    allowable_shear_stress=40  # MPa
)
```

#### 许用弯矩计算

```python
allowable_moment = calc.calculate_allowable_bending_moment(diameter)
```

### 2.5 键槽影响计算

```python
diameter = 30.0
key_slot_count = 1      # 1=单键槽，2=双键槽
key_slot_depth = 5.0    # mm

# 当量直径（考虑键槽削弱）
d_eq = calc.equivalent_diameter(
    diameter=diameter,
    key_slot_count=key_slot_count,
    key_slot_depth=key_slot_depth
)
```

### 2.6 完整示例

```python
from calculations.shaft import ShaftCalculator, calculate_torque_from_power

# 1. 已知电机参数
power = 5.5       # kW
speed = 1450      # rpm

# 2. 计算扭矩
torque = calculate_torque_from_power(power, speed)

# 3. 估算轴径
calc = ShaftCalculator(material="45")
diameter = calc.estimate_diameter_by_torsion(torque)

# 4. 校核承载能力
allowable_torque = calc.calculate_allowable_torque(diameter)

print(f"功率 {power}kW @ {speed}rpm")
print(f"扭矩 {torque:.2f} N·mm")
print(f"估算轴径 {diameter} mm（已圆整到GB/T 2822）")
print(f"许用扭矩 {allowable_torque:.2f} N·mm")
```

运行测试验证：`python tests/calculations/test_shaft_calculator.py`

## 三、GB 标准数据查询（data_loader）

`data_loader` 提供统一的标准数据读取接口，数据存储在 `src/data/standards/` 下。

### 3.1 标准尺寸系列（GB/T 2822）

```python
from standards.data_loader import get_standard_sizes, get_machining_allowance

# 第一系列（优先选用）
sizes_first = get_standard_sizes("first")

# 第二系列
sizes_second = get_standard_sizes("second")

# 全部系列
sizes_all = get_standard_sizes("all")
```

### 3.2 加工余量查询

```python
# 粗加工余量
allowance_rough = get_machining_allowance("rough")

# 半精加工余量
allowance_semi = get_machining_allowance("semi_finish")

# 精加工余量
allowance_finish = get_machining_allowance("finish")
```

### 3.3 标准件数据加载

```python
from standards.data_loader import (
    load_gb_t1096,   # 平键 GB/T 1096
    load_gb_t894,    # 弹性挡圈 GB/T 894.1
    load_gb_t812,    # 圆螺母 GB/T 812
    load_gb_t196,    # 螺纹 GB/T 196
    load_gb_t2822,   # 标准尺寸 GB/T 2822
)

# 加载平键标准
keys = load_gb_t1096()
# 返回结构：{轴径范围: {键宽, 键高, ...}}

# 加载弹性挡圈标准
circlips = load_gb_t894()
```

### 3.4 测试验证

```bash
python tests/standards/test_data_loader.py
```

## 四、SOLIDWORKS 自动化建模

`SolidWorksAdapter` 封装 SW COM API，提供 Python 可调用的建模工具。

### 4.1 启动与退出

```python
from adapters.solidworks import SolidWorksAdapter

sw = SolidWorksAdapter(sw_version="2023")

# 启动 SW
success = sw.start(visible=True, user_control=True)

try:
    # ... 建模操作 ...
    pass
finally:
    # 确保退出
    sw.quit()
```

### 4.2 文档操作

```python
# 新建零件
sw.new_part()

# 打开已有零件
sw.open_part(r"C:\path\to\part.sldprt")

# 保存
sw.save_as(r"C:\output\shaft.sldprt")
```

### 4.3 单步指令测试

对于 SW API 单步指令，可运行单步测试脚本：

```bash
python tests/adapters/test_sw_adapter_single_step.py
```

该脚本验证 SW API 的基础操作：草图创建、特征建模、保存等。

### 4.4 常见问题

**Q: comtypes 加载 sldworks.tlb 失败？**
A: 设置环境变量 `PYTHONUTF8=1` 后重试。中文 Windows 可能遇到编码问题。

**Q: pywin32 Dispatch 报"找不到 SldWorks.Application"？**
A: 先打开一次 SOLIDWORKS GUI 让它注册 COM 组件，或以管理员身份运行 Python。

## 六、模块导入路径

所有模块位于 `src/` 下，使用前需将 `src` 加入 Python 路径：

```python
import sys, os
sys.path.insert(0, os.path.join(os.getcwd(), 'src'))

# 然后即可导入
from calculations.shaft import ShaftCalculator
from standards.data_loader import load_gb_t1096, get_standard_sizes
from adapters.solidworks import SolidWorksAdapter
```

### 模块对应关系

| 功能 | 模块路径 | 主要类/函数 |
|------|---------|------------|
| 轴强度计算 | `calculations.shaft` | `ShaftCalculator`, `ShaftSection` |
| 功率/扭矩计算 | `calculations.power` | `calculate_torque_from_power` |
| 轴承计算 | `calculations.bearing` | 轴承寿命计算 |
| 标准数据加载 | `standards.data_loader` | `load_gb_t1096`, `get_standard_sizes` |
| GB 标准表 | `standards.gb_tables` | GB 标准尺寸查询 |
| 材料属性 | `standards.material_properties` | 材料数据 |
| SW 适配器 | `adapters.solidworks` | `SolidWorksAdapter` |
| comtypes 桥接 | `adapters.comtypes_bridge` | `get_feature_manager_vtable` |
| 模板管理 | `adapters.template_manager` | SW 模板配置 |
| 版本检测 | `adapters.version_detect` | SW 版本检测 |

## FAQ

**Q: 为什么不开发 GUI？**
A: MVP 阶段专注于工具层封装，面向 LLM Agent 调用。GUI 将在核心功能稳定后开发。

**Q: 是否支持 Linux/macOS？**
A: 不支持。SW COM 接口仅 Windows 可用，需 Windows 10/11 + SOLIDWORKS。

**Q: 数据文件在哪里？**
A: `src/data/standards/` 下，随包分发。不需要额外下载。
