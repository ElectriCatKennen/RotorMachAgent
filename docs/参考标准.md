## 一、GB 标准数据

### 1.1 已集成的标准数据

| 标准号 | 标准名称 | 数据文件 | 应用场景 |
|--------|---------|---------|---------|
| GB/T 1096 | 普通型 平键 | `GB_T1096_平键.json` | 键槽设计、键选型 |
| GB/T 894.1 | 轴用弹性挡圈 | `GB_T894_1_弹性挡圈.json` | 轴向定位、挡圈选型 |
| GB/T 812 | 圆螺母 | `GB_T812_圆螺母.json` | 轴向锁紧、圆螺母选型 |
| GB/T 196 | 普通螺纹 基本尺寸 | `GB_T196_螺纹.json` | 螺纹设计、紧固件选型 |
| GB/T 2822 | 标准尺寸 | `GB_T2822_标准尺寸.json` | 轴径圆整、标准尺寸选用 |
| GB/T 321 | 优先数和优先数系 | `GB_T321_优先数系.json` | 参数系列化设计 |
| GB 3100 | 国际单位制及其应用 | `GB_3100_单位换算.json` | 单位转换 |

### 1.2 数据存储位置

```
src/data/standards/
├── GB_T1096_平键.json
├── GB_T894_1_弹性挡圈.json
├── GB_T812_圆螺母.json
├── GB_T196_螺纹.json
├── GB_T2822_标准尺寸.json
├── GB_T321_优先数系.json
├── GB_3100_单位换算.json
└── 力学_常数.json
```

### 1.3 数据加载方式

```python
from standards.data_loader import (
    load_gb_t1096,    # 平键
    load_gb_t894,     # 弹性挡圈
    load_gb_t812,     # 圆螺母
    load_gb_t196,     # 螺纹
    load_gb_t2822,    # 标准尺寸
    get_standard_sizes,
    get_machining_allowance,
)

# 查询标准直径系列
sizes = get_standard_sizes("first")  # 第一系列

# 查询加工余量
allowance = get_machining_allowance("rough")
```

## 四、数据文件命名规范

### 4.1 命名规则

| 文件类型 | 命名格式 | 示例 |
|---------|---------|------|
| GB 标准 | `GB_标准号_标准中文名.json` | `GB_T1096_平键.json` |
| 通用数据 | `类别_名称.json` | `力学_常数.json` |

### 4.2 注意事项

- 标准号中的连字符使用长破折号 `—`（非短横线 `-`）
- 标准号中的点号保留（如 `GB/T 894.1`）
- 文件名使用中文名，便于识别

## 五、数据使用说明

### 5.1 加载机制

- 数据加载器（`data_loader.py`）提供统一读取接口
- 内置缓存机制，避免重复加载
- 数据文件随包分发，不需要额外下载

### 5.2 扩展标准数据

如需添加新的 GB 标准数据：

1. 在 `src/data/standards/` 下创建 JSON 文件
2. 遵循命名规范（`GB_T标准号_中文名.json`）
3. 在 `data_loader.py` 中添加加载函数
4. 编写测试验证数据完整性

### 5.3 验证数据

```bash
python tests/standards/test_data_loader.py
```

## FAQ

**Q: 标准数据是否完整？**
A: 已集成常用轴系设计相关标准。其他标准（如齿轮精度、皮带选型）待补充。

**Q: 是否支持 ISO/DIN 标准？**
A: 当前以 GB 标准为主。ISO/DIN 标准可参照相同结构扩展。

**Q: 标准数据是否公开？**
A: 标准数据本身是公共知识，已整合到代码数据库中，随包分发。机械设计手册原文（PDF）不公开，涉及知识产权。

**Q: 如何获取最新版标准？**
A: GB 标准更新后，更新对应 JSON 文件即可。数据与代码分离，便于维护。

**Q: 是否支持自定义材料？**
A: 支持。在 `ShaftCalculator._get_material_properties()` 中添加材料属性，或扩展 `materials/steel.json`。
