# 应用场景

> RotorMachAgent 面向旋转机械设计场景，本文档介绍典型应用场景与使用示例。

---

## 一、适用领域

### 1.1 目标设备

| 设备类型 | 典型部件 | 设计重点 |
|---------|---------|---------|
| 化工搅拌器 | 搅拌轴、桨叶、联轴器 | 扭矩传递、轴径估算、强度校核 |
| 泵 | 泵轴、叶轮、轴承 | 弯扭组合、临界转速 |
| 齿轮箱 | 齿轮轴、轴承、箱体 | 多级传动、齿轮啮合 |
| 涡轮机械 | 涡轮轴、叶片 | 高转速、疲劳强度 |

### 1.2 核心能力

- **轴系设计**：从功率/转速推算轴径，考虑键槽削弱、标准直径圆整
- **强度校核**：许用扭矩/弯矩计算，弯扭组合应力校核
- **标准件查询**：GB 标准平键、挡圈、圆螺母、螺纹、轴承尺寸
- **SW 建模**：参数化生成 SW 原生零件（.sldprt），保留特征树

## 二、场景一：搅拌器轴径设计

### 2.1 问题描述

已知：电机功率 5.5 kW，转速 1450 rpm，材料 45 钢，设计搅拌器传动轴。

### 2.2 解决方案

```python
import sys, os
sys.path.insert(0, os.path.join(os.getcwd(), 'src'))

from calculations.shaft import ShaftCalculator, calculate_torque_from_power

# 1. 计算输入扭矩
power = 5.5      # kW
speed = 1450     # rpm
torque = calculate_torque_from_power(power, speed)
# 结果：约 36200 N·mm

# 2. 估算轴径（按扭转强度）
calc = ShaftCalculator(material="45")
diameter = calc.estimate_diameter_by_torsion(
    torque=torque,
    allowable_shear_stress=40  # MPa
)
# 结果：自动圆整到 GB/T 2822 标准直径

# 3. 校核许用扭矩
allowable_torque = calc.calculate_allowable_torque(diameter)
print(f"轴径 {diameter}mm，许用扭矩 {allowable_torque:.2f} N·mm")
```

### 2.3 输出说明

- 估算轴径会自动圆整到 GB/T 2822 第一系列
- 许用扭矩基于材料剪切强度的 1/3 作为许用应力
- 如需更精确校核，考虑弯扭组合（见场景二）

## 三、场景二：弯扭组合轴径设计

### 3.1 问题描述

已知：扭矩 100000 N·mm，弯矩 50000 N·mm，材料 40Cr，按弯扭组合强度设计。

### 3.2 解决方案

```python
from calculations.shaft import ShaftCalculator

calc = ShaftCalculator(material="40Cr")
diameter = calc.estimate_diameter_by_bending_torsion(
    torque=100000,        # N·mm
    bending_moment=50000, # N·mm
    allowable_stress=60   # MPa
)

# 40Cr 材料属性：屈服强度 550 MPa，抗拉强度 980 MPa
```

### 3.3 适用场景

- 转轴（同时承受弯矩和扭矩）
- 悬臂轴（带皮带轮或链轮）
- 齿轮轴（齿轮啮合产生径向力）

## 四、场景三：标准件选型

### 4.1 问题描述

已知轴径 30 mm，选择配套的平键、弹性挡圈。

### 4.2 解决方案

```python
from standards.data_loader import load_gb_t1096, load_gb_t894

# 1. 查询平键尺寸（GB/T 1096）
keys = load_gb_t1096()
# 根据轴径范围查找对应键宽、键高

# 2. 查询弹性挡圈尺寸（GB/T 894.1）
circlips = load_gb_t894()
# 根据轴径查找挡圈规格
```

### 4.3 支持的标准件

| 标准件 | 标准号 | 数据文件 |
|--------|--------|---------|
| 普通平键（A/B/C型） | GB/T 1096 | `GB_T1096_平键.json` |
| 轴用弹性挡圈 | GB/T 894.1 | `GB_T894_1_弹性挡圈.json` |
| 圆螺母 | GB/T 812 | `GB_T812_圆螺母.json` |
| 普通螺纹 | GB/T 196 | `GB_T196_螺纹.json` |
| 标准尺寸系列 | GB/T 2822 | `GB_T2822_标准尺寸.json` |
| 优先数系 | GB/T 321 | `GB_T321_优先数系.json` |

详见 [参考标准.md](参考标准.md)。

## 五、场景四：SOLIDWORKS 参数化建模

### 5.1 问题描述

将设计好的轴参数化为 SW 原生零件，保留特征树可继续编辑。

### 5.2 解决方案

```python
from adapters.solidworks import SolidWorksAdapter

sw = SolidWorksAdapter(sw_version="2023")
sw.start(visible=True)

try:
    # 1. 新建零件
    sw.new_part()
    
    # 2. 参数化建模（草图 + 特征）
    # 详见 tests/adapters/test_sw_adapter_single_step.py
    
    # 3. 保存为 SW 原生文件
    sw.save_as(r"D:\output\shaft.sldprt")
finally:
    sw.quit()
```

### 5.3 关键特性

- **SW 原生格式**：输出 .sldprt，保留草图、特征、约束
- **可继续编辑**：工程师可在 SW 中二次开发
- **不依赖中间格式**：不走 STEP/IGES 路线，避免特征丢失
- **中文版 SW 适配**：支持中文基准面命名（前视基准面等）

## 六、场景五：键槽影响分析

### 6.1 问题描述

轴径 30 mm，单键槽深度 5 mm，计算当量直径评估强度削弱。

### 6.2 解决方案

```python
from calculations.shaft import ShaftCalculator

calc = ShaftCalculator()
d_eq = calc.equivalent_diameter(
    diameter=30.0,
    key_slot_count=1,      # 单键槽
    key_slot_depth=5.0     # mm
)
# 公式：d_eq = d · (1 - 0.2 · t1/d)

# 双键槽
d_eq2 = calc.equivalent_diameter(
    diameter=30.0,
    key_slot_count=2,
    key_slot_depth=5.0
)
# 公式：d_eq = d · (1 - 0.25 · t1/d)
```

### 6.3 应用场景

- 轴上开键槽后的强度评估
- 双键槽（相隔 180°）的强度削弱
- 花键轴的当量直径估算

## 七、典型工作流

### 7.1 从功率到建模的完整流程

```
1. 输入参数
   功率、转速、材料、工况
        ↓
2. 扭矩计算
   calculate_torque_from_power()
        ↓
3. 轴径估算
   estimate_diameter_by_torsion()
   → 自动圆整到 GB/T 2822
        ↓
4. 强度校核
   calculate_allowable_torque()
   → 验证是否满足工况
        ↓
5. 标准件选型
   load_gb_t1096() 等
   → 选择键、挡圈、螺母
        ↓
6. SW 建模
   SolidWorksAdapter
   → 生成 .sldprt 文件
```

### 7.2 执行示例

```bash
# 1. 验证计算模块
python tests/calculations/test_shaft_calculator.py

# 2. 验证标准数据
python tests/standards/test_data_loader.py

# 3. SW 建模测试（需 SW 环境）
python tests/adapters/test_sw_adapter_single_step.py

# 4. 完整轴建模
python tests/test_shaft_modeling.py
```

## 八、行业应用

### 8.1 锂电池浆料搅拌设备

- **场景**：搅拌轴设计、扭矩校核、桨叶参数化
- **特点**：高粘度流体，扭矩大，需考虑疲劳强度
- **应用**：轴径估算、许用扭矩校核

### 8.2 化工泵设计

- **场景**：泵轴设计、轴承选型、密封设计
- **特点**：需考虑弯扭组合（叶轮重量 + 径向力）
- **应用**：弯扭组合估算、临界转速计算

### 8.3 通用减速器设计

- **场景**：多级传动轴系设计
- **特点**：多级齿轮传动，扭矩逐级变化
- **应用**：各级轴径估算、强度校核

## FAQ

**Q: 是否支持非圆形截面轴？**
A: 当前版本仅支持圆形截面。非圆形截面需自行实现截面系数计算。

**Q: 是否支持空心轴？**
A: 计算模块支持空心轴（通过参数指定内径），建模模块待开发。

**Q: 是否支持有限元分析？**
A: 不直接支持。输出 SW 原生文件后，可在 ANSYS/Abqus 中进行 FEA。

**Q: 如何扩展支持新材料？**
A: 在 `ShaftCalculator._get_material_properties()` 中添加材料属性即可。

**Q: SW 建模是否支持装配体？**
A: 当前版本聚焦零件建模，装配体功能待开发。
